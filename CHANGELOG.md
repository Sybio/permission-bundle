## [1.0.0] - 2025-12-19

### Added
- Initial release of the Permission Bundle
- Value Object-based permission system
- Integration with Symfony Security (Voter)
- Integration with Symfony Validator
- Auto-configuration for permission checkers
- PermissionValidatorInterface service
- PermissionConstraintValidator for constraint validation
- PermissionVoter for security integration
